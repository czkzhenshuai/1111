# 贪吃蛇游戏 (Snake Game)

一个用 Python + Pygame 开发的经典贪吃蛇游戏。

## 功能特性

- 🎮 使用方向键控制蛇的移动
- 🍎 吃掉食物来增长身体
- 💥 碰撞检测（撞墙或咬到自己会游戏结束）
- 📊 实时显示分数
- 🎯 难度逐渐升级（速度会增加）

## 安装依赖

```bash
pip install pygame
```

## 运行游戏

```bash
python snake_game.py
```

## 游戏操作

- **方向键** (↑ ↓ ← →)：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **ESC**：退出游戏
- **R**：游戏结束后重新开始

## 游戏规则

1. 控制蛇去吃食物（红色方块）
2. 每吃一个食物，蛇会变长，分数加10分
3. 蛇不能碰到墙壁或咬到自己，否则游戏结束
4. 分数越高，蛇移动的速度越快

## 游戏窗口

- 分辨率：800x600 像素
- 网格大小：20x20 像素
- 帧率：60 FPS

## 代码结构

- `snake_game.py`：主游戏文件，包含所有游戏逻辑
